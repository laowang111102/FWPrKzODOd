# 基于Python Django的图书馆管理系统 python288

## 项目概述

本项目是基于Python语言和Django框架开发的图书馆管理系统，支持图书馆的日常管理功能，为管理员和用户提供了一个便捷、高效的操作平台。

## 技术栈

- **开发语言**：Python
- **开发框架**：Django
- **数据库**：MySQL
- **架构**：B/S架构
- **前端技术**：Vue

## 功能模块

### 前台功能

- 首页
- 图书馆介绍
- 图书信息
- 公告信息
- 留言板
- 个人中心
- 后台管理入口

### 后台功能

#### 管理员功能

- 用户管理
- 图书馆管理
- 图书信息管理
- 图书类型管理
- 借阅信息管理
- 归还信息管理
- 续借信息管理
- 罚金信息管理
- 留言板管理
- 系统管理

#### 用户功能

- 个人信息管理
- 借阅信息管理
- 归还信息管理
- 续借信息管理
- 罚金信息管理
- 留言板管理
- 收藏管理

## 系统角色

- 管理员：负责图书馆的日常管理和用户管理。
- 用户：可查询图书信息、借阅图书、管理个人信息等。
